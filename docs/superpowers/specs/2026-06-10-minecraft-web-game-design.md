# 网页版「我的世界」游戏设计文档

日期：2026-06-10
状态：已确认

## 目标

构建一个可以**双击 index.html 直接运行**的浏览器版 Minecraft 风格体素游戏。无需安装、无需构建工具、无需服务器、离线可玩。

## 需求范围

用户确认的三项关键决策：

1. **平台**：浏览器网页版（Three.js + WebGL）
2. **功能**：经典完整版 —— 随机地形生成（山地、树木）、第一人称行走跳跃、挖方块/放方块、多种方块类型、快捷栏切换
3. **世界**：无限世界（区块动态加载）

明确不做（本期范围外）：昼夜循环、水体、存档读档、背包合成、生物、移动端触屏适配。

## 技术方案

方案 A（已确认）：Three.js 负责渲染，自研体素引擎负责游戏逻辑。

- Three.js 使用 UMD 版 `three.min.js` 存放在本地 `lib/` 目录，通过经典 `<script>` 标签加载，保证 `file://` 协议下双击可运行且离线可用
- 所有方块贴图用 canvas 程序化生成像素风纹理，项目零图片素材
- 全部代码为纯静态文件，无 npm、无打包器

## 文件结构

```
cc-myworld/
├── index.html          入口页面：canvas、UI 元素、script 标签按序加载
├── lib/three.min.js    Three.js 本地副本
└── js/
    ├── noise.js        柏林噪声实现（含 fBm 多层叠加）
    ├── blocks.js       方块类型注册表 + canvas 程序化贴图图集
    ├── world.js        区块存储（Map）+ 无限地形生成 + 方块读写接口
    ├── mesher.js       区块网格构建：面剔除、图集 UV、顶点明暗/AO
    ├── player.js       玩家物理：WASD/跳跃/重力/AABB 碰撞、视角控制
    ├── interact.js     DDA 体素射线：选中高亮、左键挖、右键放
    ├── ui.js           快捷栏（含方块图标）、准星、提示遮罩
    └── main.js         游戏主循环、按玩家位置动态加载/卸载区块
```

各模块通过全局命名空间（如 `window.Game.*`）通信，依赖顺序由 index.html 中 script 标签顺序保证。

## 核心机制设计

### 区块与无限世界

- 区块尺寸 16×16 列 × 高 64 格，方块数据存 `Uint8Array`（16×64×16）
- 区块以 `"cx,cz"` 字符串为键存入 `Map`
- 玩家移动时，以玩家所在区块为中心、半径 4 个区块内的区块自动生成；超出半径 +2 的区块卸载（释放网格，保留已修改的方块数据）
- 每帧最多生成/构网格 1~2 个新区块，避免掉帧
- 远景雾颜色与天空一致，遮蔽加载边界

### 地形生成

- 固定随机种子（页面刷新换新世界），同一种子下区块生成结果确定，走回头路地形不变
- 高度图 = 多层柏林噪声（fBm）：低频大起伏（山脉/平原）+ 高频细节
- 分层：表层草方块、下面 3 格泥土、再往下石头；低洼区域（接近最低高度）表层换沙子
- 树木：按噪声+伪随机散布在草地上，原木 4~6 格高，顶部 5×5×3 树叶簇；树可跨区块边界（通过世界坐标写入接口处理）

### 方块系统（8 种）

| ID | 名称 | 贴图特征 |
|----|------|---------|
| 1 | 草方块 | 顶面绿、侧面土+草边、底面土 |
| 2 | 泥土 | 棕色噪点 |
| 3 | 石头 | 灰色噪点 |
| 4 | 原木 | 侧面树皮纹、顶底年轮 |
| 5 | 木板 | 横条纹木色 |
| 6 | 树叶 | 深绿噪点 |
| 7 | 沙子 | 米黄噪点 |
| 8 | 砖块 | 红砖+灰缝 |

贴图为 16×16 像素/格，拼成一张 canvas 图集，`NearestFilter` 保持像素锐利。

### 渲染与性能

- 每区块所有方块合并为一个 `BufferGeometry`，仅生成与空气（或透明方块）相邻的面
- 顶点着色实现方向明暗（顶 1.0 / 侧 0.8、0.6 / 底 0.5）+ 简单顶点 AO（角落变暗）
- 挖/放方块仅重建所在区块网格；若方块位于区块边界，同时重建相邻区块

### 玩家物理

- 碰撞箱 0.6×1.8×0.6（宽×高×深），视点高 1.62
- 重力 + 跳跃初速度，参数对标 MC 手感；逐轴碰撞解析（先 Y 后 X/Z）防穿墙
- 移动：WASD 相对视角方向，空格跳（仅落地时）
- 视角：Pointer Lock API，鼠标控制 yaw/pitch，pitch 限制 ±89°
- 掉出世界底部（y < -10）自动传送回出生点

### 交互

- 准星射线：DDA 体素遍历算法，最大距离 6 格，精确返回命中方块与命中面
- 左键挖除方块，右键在命中面相邻格放置当前快捷栏方块
- 放置前检测与玩家 AABB 相交则禁止（不能把自己埋进方块）
- 选中方块显示黑色线框高亮

### UI

- 屏幕中央十字准星（CSS 绘制）
- 底部快捷栏 8 格，图标取自贴图图集 canvas；数字键 1-8 或滚轮切换，选中格高亮
- 初始遮罩：操作说明 +「点击开始」；ESC 释放鼠标后重新显示遮罩

## 错误处理

- WebGL 不可用：显示中文提示信息而非白屏
- Pointer Lock 请求失败/被释放：显示遮罩等待用户重新点击
- 玩家掉出世界：传送回出生点

## 验证标准

构建完成后在真实浏览器中打开游戏逐项验证：

1. 页面打开即见地形（山地起伏、有树、低洼有沙）
2. WASD 行走、空格跳跃、鼠标转视角，不穿墙不掉地底
3. 左键可挖掉方块，右键可放置方块，线框高亮正确
4. 数字键/滚轮切换快捷栏 8 种方块，放出来的方块与选择一致
5. 朝一个方向持续行走，新区块持续生成，无明显卡顿，走回头路地形不变
6. ESC 后点击可恢复游戏
7. 浏览器控制台无报错
