# PvP + 组队系统 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 玩家可用剑/弓互相攻击（PvP），并可通过邀请/接受组成最多 4 人队伍实现友伤豁免与 64 格范围内打怪 XP 平分。

**Architecture:** 纯客户端 + Cloudflare Durable Object 扩展，零新文件。服务器内存维护 `teams` Map 和 `pendingInvites` Map，不持久化。PvP 伤害复用现有 `damagePlayer`；XP 分配扩展现有 `grantKill`；E 键上下文感知（面向玩家→邀请，面向 NPC→任务）。

**Tech Stack:** Cloudflare Durable Object（WorldDO），HTML5 Canvas，Three.js，现有 IIFE 模块模式

**Spec:** `docs/superpowers/specs/2026-06-15-pvp-team-design.md`

  removeFromTeam(s) {
    if (!s || s.teamId === null) return;
    const teamId = s.teamId;
    const team = this.teams.get(teamId);
    s.teamId = null;
    if (!team) return;
    team.members.delete(s.pid);
    if (team.members.size <= 1) {
      // 仅剩 0 或 1 人：解散
      for (const mpid of team.members) {
        const [mws, ms] = this.sessionByPid(mpid);
        if (ms) ms.teamId = null;
        if (mws) this.send(mws, { t: 'teamUpdate', members: [], leaderPid: null });
      }
      this.teams.delete(teamId);
      return;
    }
    // 队长离队：转让给 pid 最小的在线成员
    if (team.leaderPid === s.pid) {
      team.leaderPid = Math.min(...team.members);
    }
    this.broadcastTeamUpdate(teamId);
  }

  // 向队伍所有在线成员广播当前队伍状态
  broadcastTeamUpdate(teamId) {
    const team = this.teams.get(teamId);
    if (!team) return;
    const members = [];
    for (const mpid of team.members) {
      const [, ms] = this.sessionByPid(mpid);
      if (ms) members.push({ pid: mpid, name: ms.name });
    }
    const msg = { t: 'teamUpdate', members, leaderPid: team.leaderPid };
    for (const mpid of team.members) {
      const [mws] = this.sessionByPid(mpid);
      if (mws) this.send(mws, msg);
    }
  }

  onTeamInvite(ws, s, msg) {
    if (s.dead) return;
    const [tws, ts] = this.sessionByPid(msg.pid);
    if (!ts || !tws) return; // 对方不在线
    // 对方满队
    if (ts.teamId !== null) {
      const team = this.teams.get(ts.teamId);
      if (team && team.members.size >= 4) { this.send(ws, { t: 'teamErr', reason: 'full' }); return; }
    }
    // 惰性清理：覆盖旧邀请（后邀作废旧邀）
    this.pendingInvites.set(msg.pid, { fromPid: s.pid, expiresAt: Date.now() + 30000 });
    this.send(tws, { t: 'teamInviteFrom', pid: s.pid, name: s.name });
  }

  onTeamAccept(ws, s, msg) {
    const invite = this.pendingInvites.get(s.pid);
    if (!invite || invite.fromPid !== msg.pid || Date.now() > invite.expiresAt) {
      this.send(ws, { t: 'teamErr', reason: 'no_invite' }); return;
    }
    this.pendingInvites.delete(s.pid);
    const [, inviterS] = this.sessionByPid(invite.fromPid);
    if (!inviterS) return; // 邀请者已离线
    let team;
    if (inviterS.teamId !== null) {
      team = this.teams.get(inviterS.teamId);
      if (!team || team.members.size >= 4) { this.send(ws, { t: 'teamErr', reason: 'full' }); return; }
    } else {
      // 新建队伍
      const teamId = this.nextTeamId++;
      team = { id: teamId, leaderPid: invite.fromPid, members: new Set([invite.fromPid]) };
      this.teams.set(teamId, team);
      inviterS.teamId = teamId;
    }
    team.members.add(s.pid);
    s.teamId = team.id;
    this.broadcastTeamUpdate(team.id);
  }

  onTeamDecline(ws, s, msg) {
    const invite = this.pendingInvites.get(s.pid);
    if (invite && invite.fromPid === msg.pid) this.pendingInvites.delete(s.pid);
  }

  onTeamLeave(ws, s) {
    this.removeFromTeam(s);
  }
